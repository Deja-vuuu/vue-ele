### 项目介绍
 
 1. 此项目为外卖App核心的商家模块的SPA
 2. 包括商品、评论、商家介绍、购物车等多个子模块
 3. 使用Vuejs全家桶+ES6+Webpack等前端最新最热的技术	
 4. 采用模块化、组件化、工程化的模式开发
